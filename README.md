# Homebrew Smelter

Official Homebrew tap for [Smelter](https://github.com/YOURUSERNAME/smelter) - the type-safe scripting language with 41ms startup.

## Installation
```bash
brew tap YOURUSERNAME/smelter
brew install smelter
```

## What You Get

- `smt` command - The Smelter interpreter
- 41ms startup time (faster than Ruby!)
- Full type safety at compile time
- 9.8MB self-contained binary

## Quick Test
```bash
smt eval '(+ 1 2)'  # Returns: 3
```
